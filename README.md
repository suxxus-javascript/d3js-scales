# d3js-scales (studies)
module d3js (Encodings that map abstract data to visual representation)

## Clone the repo & install
```
git clone https://github.com/suxxus-studies/d3js-scales.git
cd ./d3js-scales
npm install
```
## Usage
npm run dev
